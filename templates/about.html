The Application consists of 2 parts :.

Web Application and Chrome Extension

  • Web Application :Through Web Application we can get detailed location (address, city, state, country), registrar, URL existence period (age), Alexa Rank, and IP information. Along with this, detailed results, user’s browsing history, live analysis of phishing activity detected through our application is maintained in Web Application.

  • Chrome Extension :Using JavaScript in Extension an automated request is sent to our Web Application Endpoint in Real-Time that returns a JSON response and through which a URL’s safety status is detected. If user visits an unsafe URL he/she is alerted through the Chrome Extension about Phishing URL, and is redirected to a safe page. Browser extension is developed for seamless and safe surfing on browsers. Our extension support Chrome, and Brave most trending browsers.
